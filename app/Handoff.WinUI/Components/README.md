# UI Components

Place reusable WinUI 3 UI building blocks here.

- `Controls`: small reusable controls, usually backed by `UserControl`.
- `Layouts`: structural components such as shells, navigation frames, and reusable page layouts.
- `Dialogs`: dialogs, flyouts, popups, and modal views.

Keep feature-specific screens in `Pages` unless the UI is reusable across multiple screens.
